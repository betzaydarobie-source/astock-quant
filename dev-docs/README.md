# dev-docs —— 开发 / 设计存档

这个文件夹收录的是 astock-quant 项目**开发过程中的技术设计文档**。它们记录了
项目早期「短期涨跌预测系统」的架构、因子库、模型训练、回测等设计细节，是开发
历史的存档，**不是当前用户文档**。

> ⚠️ **重要说明**：本项目已在 Stage 5 转向「中长期价值选股」（详见仓库根目录
> `README.md` 与 `人话报告/12-Stage5价值选股改造.md`）。下面 P0–Stage4 阶段的
> 文档大多围绕**已被弃用的短期涨跌预测系统**，仅作开发存档保留。读它们时请记得
> 这一背景 —— 当前系统的真实状态以根目录 `README.md` 为准。
>
> 想看大白话讲解，请看仓库的 `人话报告/` 文件夹（面向项目主人的通俗版，当前有效）。
